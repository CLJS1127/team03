🌉 CareerBridge 웹사이트 시작 가이드

===============================================
✅ 완성된 페이지 목록
===============================================

1. index.html - 메인 페이지
   - 히어로 섹션
   - 체험 프로그램 미리보기
   - 주요 기능 소개
   - 성공 사례
   - 제휴 기업

2. programs.html - 프로그램 목록 페이지
   - 필터 사이드바 (날짜, 직무, 지역, 인턴전환, 기업규모)
   - 정렬 기능
   - 페이지네이션

3. program-detail.html - 프로그램 상세 페이지
   - 프로그램 일정 타임라인
   - FAQ 아코디언
   - 관심 프로그램 등록
   - 공유하기 기능

4. application.html - 신청 페이지
   - 입력 폼 유효성 검사
   - 파일 업로드
   - 글자 수 카운터
   - 약관 동의

5. mypage.html - 마이페이지
   - 신청 내역 (심사중/승인/거절)
   - 관심 프로그램
   - 완료 프로그램
   - 인증서
   - 설정

6. login.html - 로그인 페이지
   - 이메일/비밀번호 로그인
   - 소셜 로그인 (카카오, 네이버, 구글)

7. signup.html - 회원가입 페이지
   - 회원가입 폼
   - 비밀번호 강도 체크
   - 이메일 중복 확인
   - 관심 분야 선택

===============================================
🚀 실행 방법
===============================================

1. index.html 파일을 더블클릭하세요
   (또는 브라우저로 드래그해서 열기)

2. 페이지 이동 경로:
   메인 → 체험 프로그램 → 상세 → 신청 → 마이페이지

3. 모든 버튼과 링크가 작동합니다!

===============================================
💡 주요 기능
===============================================

✅ 완전한 반응형 디자인 (모바일/태블릿/데스크톱)
✅ 부드러운 애니메이션 효과
✅ FAQ 아코디언 토글
✅ 파일 업로드 (10MB 제한)
✅ 폼 유효성 검사
✅ 비밀번호 강도 체크
✅ 탭 네비게이션
✅ 모달 다이얼로그
✅ 필터링 및 정렬
✅ 관심 프로그램 저장
✅ 로고 디자인 적용
